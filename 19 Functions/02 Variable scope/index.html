Variable Scope in Functions

What is Scope?

Scope means where in your code a variable can be accessed. Think of it like the “area” in which the variable exists and can be used.

Types of Scope

  • Global Scope: Variables declared outside of any function. They can be accessed anywhere in the code.
  • Local Scope: Variables declared within a function. They can only be accessed within that function.
  • Block Scope: Variables declared with `let` or `const` inside a block (like an `if` statement) are only accessible within that block.
