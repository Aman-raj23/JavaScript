reduce() method

The reduce() method executes a reducer function on each element of an array, resulting in a single output value.
Or,
The reduce() method reduce the elements of an array to a single value.
Itâ€™s commonly used for sums, products, averages, or building objects from arrays.

Syntax:
array.reduce((accumulator, currentValue, index, array) => { return updatedAccumulator; }, initialValue);

  • accumulator = value carried over each iteration (starts as initialValue).
  • currentValue = current element being processed.
  • initialValue = optional starting value (if not provided, it takes the first array element).
