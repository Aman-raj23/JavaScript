Variable

A variable is like a container that stores data (number, text, array, object, etc.) which we can reuse or change later

Ways to Declare Variables

JavaScript has 3 keywords for declaring variables:

  1. var - old way (not recommended now, has scope issues)
  2. let - modern, block-scoped, can be reassigned.
  3. const - modern, block-scoped, cannot be reassigned.
